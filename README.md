### 🚀 Nad czym teraz pracuję?

**[CookingStation - Autorski Silnik Renderujący 3D]**
Obecnie (w ramach projektu ZTGK) rozwijam od zera niskopoziomowy silnik gier w **C++ / OpenGL**. Skupiam się w nim przede wszystkim na wydajności i architekturze. Główne systemy, za które odpowiadam, to:
* **Architektura ECS:** Własna implementacja Entity Component System.
* **Fizyka i Raycasting:** System detekcji kolizji AABB oraz raycasting oparty.
* **Natywne Skryptowanie:** Opracowanie systemu skryptowania w C++ łączącego się z logiką encji.
* **Rendering:** Integracja warstwy OpenGL
